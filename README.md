# Решатель - PWA Версия

Интерактивное приложение для генерации случайных решений в формате Progressive Web App.

## Особенности PWA

✅ **Установка на устройство** - Может быть установлено как нативное приложение
✅ **Работа офлайн** - Полная функциональность без интернета
✅ **Push уведомления** - Поддержка уведомлений (в разработке)
✅ **Кэширование** - Быстрая загрузка за счет локального кэша
✅ **Адаптивный дизайн** - Оптимизирован для мобильных устройств

## Установка

1. Откройте приложение в браузере Chrome, Edge или Firefox
2. Нажмите на значок установки в адресной строке
3. Подтвердите установку
4. Приложение будет доступно как отдельное приложение

## Технические детали

### Файлы PWA:
- `manifest.json` - Конфигурация PWA приложения
- `sw.js` - Service Worker для офлайн работы
- `icons/` - Иконки разных размеров для установки
- `icon.svg` - Векторная иконка приложения

### Основные компоненты:
- **Монета** - Генерация орла или решки
- **Кубик** - Бросок шестигранного кубика (1-6)
- **Колесо фортуны** - Случайный выбор из списка вариантов
- **История** - Сохранение результатов последних действий
- **Настройки звука** - Переключение звуковых эффектов

## Разработка

Для запуска локального сервера:
```bash
python -m http.server 8000
```

Или используйте любой HTTP сервер:
```bash
npx serve
# или
php -S localhost:8000
```
